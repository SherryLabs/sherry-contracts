# Sherry Smart 

## Instrucciones de Despliegue

Para desplegar los contratos usando Hardhat Ignition, ejecuta el siguiente comando:

```sh
npx hardhat ignition deploy ./ignition/modules/Brand.ts --network avalancheFuji
```
